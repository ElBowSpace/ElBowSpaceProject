Welcome to the ElbowSpace User Guide. If you find yourself elsewhere on the site and want to get back, just click the 'User Guide' link in the contents box again.

As you might imagine, to make connections on this site you first need to sign up. Click the link that says sign up to get started.

We can fill out a new user we'll call 'Generic Person.' Your user name, a unique identifier, and your password are both needed to move forward. Just note, if you leave the first and last name blank people will have trouble finding you later. Click the 'Sign Up' button to continue.

If the username was unique and your password was good then you should be redirected to the ElBow Space home page. Congrats, you've signed up and if you look to contents you've also been automatically logged in. See the different options? 'Login' and 'Sign Up' were replaced with 'Logout', 'User List', and 'Post List'. Lets have a look at the User List first by clicking the link.

The User List is where you'll see your potential connections. For now the list is small and easy to navigate, but with more use a search feature will be added to help you find people. Did you notice you have already been added to the list? Click on your name to see your options.

User detail is displayed such as name and email, but if you want to do any editing you'll have to click the 'edit user' link.

Here's where you can edit your name, email, and password. Saving will log you out and return you to the home page for Elbow Space. This is a security feature that signs you out of all computers if the password changes. If you don't want to change your password at this time, hit the browser back button to return to your detail page.

Lets try making a new post for ourselves. Click the link to 'New Post' to continue.

Type in a message for all to see and hit 'Save!'

Ta-da! The line at the top there in the image below is an post detail link to let you fix spelling mistakes. It is always available to you on your own posts. You can also delete the post there.

Another feature of this link is the reply link. From there you can reply to the post with a new post. Note: deleting a post also deletes all replies to that post.

Let's return to the User List where we see another user has joined Elbow Space. Click on the link to see New Friends detail page.

The user detail options have changed. That's because only you can edit your own account info. If you want to connect with this user then you can click the make connection link. You'll be returned to the user list, but don't worry. It worked! You are connected now.

That's it for now. This tutorial will be extended as more options come online. Play around and see what you can do. Be sure to let the developers know what changes you'd like to see by logging issues on their GitHub repo.
